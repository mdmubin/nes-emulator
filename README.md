# nes-emulator

An emulator for the Nintendo Entertainment System. 

**Work in Progress**

### Features

- [x] CPU Emulation (All official Op Codes Working)
- [x] Memory
- [x] Cartridge
  [-] Memory Mapping (partial)
  - [x] Mapper - 000
  - [ ] Mapper - 037
- [ ] APU (audio)
  [*] PPU (picture / graphics)
- [ ] I/O
